# ❤️ Birthday Surprise Website

An interactive romantic birthday surprise website made using HTML, CSS, JavaScript, Canvas animations, and music.

## ✨ Features
- Animated blooming heart tree 🌸
- Floating hearts ❤️
- Romantic typewriter messages 💌
- Background birthday music 🎵
- Smooth opening animation ✨
- Responsive modern UI 💖

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery
- Canvas Animation
